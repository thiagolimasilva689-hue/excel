# 📊 Excel para Análise de Dados

Projetos e exercícios práticos de Excel aplicados à análise de dados.

## 📋 Habilidades desenvolvidas

- Fórmulas: SOMA, MÉDIA, CONT.NÚM, SE
- PROCV para busca de valores
- Tabelas dinâmicas
- Gráficos (barras, pizza, linha)
- Formatação condicional
- Análise de dados de supermercado

## 📁 Projetos

| Projeto | Descrição |
|---------|-----------|
| Controle de Estoque | Planilha com SOMA, MÉDIA, Status (SE) |
| Análise de Vendas | Tabela dinâmica + gráficos |
| Lista de Supermercado | PROCV + relatório |

## 🛠️ Ferramentas

- Microsoft Excel / Google Planilhas

## 🚀 Como usar

1. Baixe os arquivos `.xlsx` ou `.csv`
2. Abra no Excel ou Google Planilhas
